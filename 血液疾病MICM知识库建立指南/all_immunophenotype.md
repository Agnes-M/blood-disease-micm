# 急性淋巴细胞白血病(ALL)免疫表型特征

## 概述

免疫表型分析是急性淋巴细胞白血病(ALL)诊断、分类和预后评估的关键组成部分。通过流式细胞术和免疫组化技术，可以确定白血病细胞的淋巴系来源(B系或T系)、分化阶段和特定亚型。ALL的免疫表型特征具有淋巴系分化标志，但不同亚型之间存在明显差异，对治疗方案选择和预后评估具有重要意义。

## 流式细胞术标志物表达模式

### B-ALL共同标志物
- **CD19**：几乎所有B-ALL阳性(>95%)，是B系谱系特异性标志物
- **CD79a**：大多数B-ALL阳性(>90%)，是B系谱系特异性标志物
- **CD22**：大多数B-ALL阳性(>90%)，可胞质或膜表达
- **PAX5**：大多数B-ALL阳性(>90%)，是B系谱系特异性转录因子
- **CD24**：大多数B-ALL阳性(>85%)
- **TdT**：大多数B-ALL阳性(>90%)，是原始淋巴细胞标志物

### T-ALL共同标志物
- **CD3**：所有T-ALL胞质CD3阳性，是T系谱系特异性标志物
- **CD7**：几乎所有T-ALL阳性(>95%)，是T系谱系相关标志物
- **CD2**：大多数T-ALL阳性(>75%)
- **CD5**：大多数T-ALL阳性(>80%)
- **CD1a**：皮质T-ALL阳性(约60%)
- **TdT**：大多数T-ALL阳性(>90%)

### 原始细胞标志物
- **CD34**：约70%的B-ALL和50%的T-ALL阳性
- **CD38**：大多数ALL阳性，表达强度可变
- **HLA-DR**：几乎所有B-ALL阳性，大多数T-ALL阴性
- **CD45**：ALL通常弱表达或中等表达
- **TdT**：大多数ALL阳性，是原始淋巴细胞的重要标志物

### 谱系交叉表达
- **髓系标志物**：
  - CD13：约20-30%的ALL可阳性
  - CD33：约20-25%的ALL可阳性
  - CD117：极少数ALL可阳性
  - MPO：ALL通常阴性，是鉴别ALL和AML的重要标志

- **NK细胞标志物**：
  - CD56：约15-20%的T-ALL可阳性，与预后不良相关
  - CD16：极少数ALL可阳性

- **其他标志物**：
  - CD123：大多数B-ALL阳性
  - CD66c：约40%的B-ALL阳性，常见于BCR-ABL1阳性和超二倍体B-ALL

## B-ALL分化阶段和亚型免疫表型特征

### 按分化阶段分类
1. **前B细胞(Pro-B)ALL**：
   - CD19(+)，CD79a(+)，CD22(+)
   - CD10(-)，胞质μ链(-)，表面免疫球蛋白(-)
   - TdT(+)，CD34(+)，HLA-DR(+)
   - 常见于婴儿ALL和KMT2A重排ALL

2. **普通B细胞(Common-B)ALL**：
   - CD19(+)，CD79a(+)，CD22(+)
   - CD10(+)，胞质μ链(-)，表面免疫球蛋白(-)
   - TdT(+)，CD34(+/-)，HLA-DR(+)
   - 最常见的B-ALL亚型

3. **前B细胞(Pre-B)ALL**：
   - CD19(+)，CD79a(+)，CD22(+)
   - CD10(+/-)，胞质μ链(+)，表面免疫球蛋白(-)
   - TdT(+)，CD34(+/-)，HLA-DR(+)
   - 常见于儿童ALL

4. **成熟B细胞(Mature-B)ALL/Burkitt白血病**：
   - CD19(+)，CD79a(+)，CD22(+)
   - CD10(+)，表面免疫球蛋白(+)
   - TdT(-)，CD34(-)，HLA-DR(+)
   - 表面IgM(+)，轻链限制性(κ或λ)
   - 常伴有MYC重排

### 按遗传学特征分类
1. **伴t(9;22)(q34;q11.2)/BCR-ABL1的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)，CD10(+)
   - CD34(+)，TdT(+)，HLA-DR(+)
   - CD13(+)，CD33(+)，约60-70%阳性
   - CD25(+)，约70-80%阳性，是特征性表达
   - CD66c(+)，约80%阳性
   - 常见于成人ALL

2. **伴t(12;21)(p13;q22)/ETV6-RUNX1的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)，CD10(+)
   - CD34(+)，TdT(+)，HLA-DR(+)
   - CD9(+)，CD38(强+)
   - CD20(+)，约40-50%阳性
   - 常见于儿童ALL，预后良好

3. **伴t(1;19)(q23;p13.3)/TCF3-PBX1的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)，CD10(+)
   - CD34(-)，TdT(+)，HLA-DR(+)
   - CD9(+)，CD20(-)
   - CD13(+)，约30-40%阳性
   - 常见于儿童和青少年

4. **伴t(v;11q23)/KMT2A重排的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)
   - CD10(-)，是特征性表达
   - CD34(+/-)，TdT(+)，HLA-DR(+)
   - CD15(+)，CD65(+)，约60-70%阳性
   - CD133(+)，NG2(+)
   - 常见于婴儿ALL，预后不良

5. **伴超二倍体的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)，CD10(+)
   - CD34(+/-)，TdT(+)，HLA-DR(+)
   - CD123(强+)，CD66c(+)
   - CD20(-)，CD45(弱+)
   - 常见于儿童ALL，预后良好

6. **伴低二倍体的B-ALL**：
   - CD19(+)，CD79a(+)，CD22(+)，CD10(+)
   - CD34(+)，TdT(+)，HLA-DR(+)
   - CD45(+)，表达强度高于超二倍体
   - CRLF2过表达，约50%
   - 预后不良

## T-ALL分化阶段和亚型免疫表型特征

### 按分化阶段分类
1. **早期T前体细胞(ETP)ALL**：
   - 胞质CD3(+)，CD7(+)
   - CD1a(-)，CD8(-)，CD5(弱+或-)
   - CD34(+)，CD13(+)，CD33(+)，CD117(+)
   - CD4(-)，表面CD3(-)
   - 预后不良

2. **前T细胞(Pro-T)ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)
   - CD1a(-)，CD4(-)，CD8(-)
   - CD34(+/-)，TdT(+)
   - CD5(+)，表面CD3(-)

3. **前T细胞(Pre-T)ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+)
   - CD1a(-)，CD4(+)和/或CD8(+)
   - CD34(+/-)，TdT(+)
   - 表面CD3(-)

4. **皮质T细胞(Cortical-T)ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+)
   - CD1a(+)，CD4(+)和CD8(+)双阳性
   - CD34(-)，TdT(+)
   - 表面CD3(+/-)
   - 最常见的T-ALL亚型

5. **成熟T细胞(Mature-T)ALL**：
   - 胞质CD3(+)，表面CD3(+)
   - CD7(+)，CD2(+)，CD5(+)
   - CD1a(-)，CD4(+)或CD8(+)单阳性
   - CD34(-)，TdT(+/-)

### 按遗传学特征分类
1. **伴TLX1(HOX11)过表达的T-ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+)
   - CD1a(+)，CD4(+)和CD8(+)双阳性
   - 皮质T-ALL免疫表型
   - 预后相对良好

2. **伴TLX3(HOX11L2)过表达的T-ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+)
   - CD1a(+/-)，CD4(+)和/或CD8(+)
   - 预后不良

3. **伴TAL1过表达的T-ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+)
   - CD1a(+/-)，CD4(+)和/或CD8(+)
   - 常见于儿童T-ALL

4. **伴HOXA过表达的T-ALL**：
   - 胞质CD3(+)，CD7(+)，CD2(+)，CD5(+/-)
   - CD1a(-)，CD4(-)，CD8(-)
   - 早期T-ALL免疫表型
   - 预后不良

## 免疫组化标志物表达模式

### B-ALL标志物
- **CD79a**：B系谱系特异性标志物，染色定位于胞质
- **CD19**：B系谱系特异性标志物，染色定位于细胞膜
- **PAX5**：B系谱系特异性转录因子，染色定位于细胞核
- **CD10**：普通B-ALL和前B-ALL阳性，染色定位于细胞膜
- **TdT**：原始淋巴细胞标志物，染色定位于细胞核
- **CD20**：部分B-ALL阳性，成熟B-ALL强阳性
- **CD34**：原始细胞标志物，染色定位于细胞膜

### T-ALL标志物
- **CD3**：T系谱系特异性标志物，染色定位于胞质和/或细胞膜
- **CD7**：T系谱系相关标志物，染色定位于细胞膜
- **CD5**：T系谱系相关标志物，染色定位于细胞膜
- **CD1a**：皮质T-ALL标志物，染色定位于细胞膜
- **TdT**：原始淋巴细胞标志物，染色定位于细胞核
- **CD4**：辅助T细胞标志物，染色定位于细胞膜
- **CD8**：细胞毒T细胞标志物，染色定位于细胞膜

## 微小残留病变(MRD)监测标志物

### B-ALL MRD监测
- **白血病相关免疫表型(LAIP)**：
  - CD19+/CD10+/CD34+/CD38弱+
  - CD19+/CD10+/CD20异常表达
  - CD19+/CD10+/CD45弱+
  - CD19+/CD34+/CD38弱+/CD58强+
  - CD19+/CD34+/CD13+和/或CD33+
  - CD19+/CD10+/CD66c+

- **常用MRD标志物组合**：
  - CD19/CD10/CD34/CD38/CD45
  - CD19/CD10/CD20/CD45
  - CD19/CD34/CD38/CD58
  - CD19/CD34/CD13/CD33
  - CD19/CD10/CD66c

### T-ALL MRD监测
- **白血病相关免疫表型(LAIP)**：
  - 胞质CD3+/CD7+/表面CD3-
  - 胞质CD3+/CD7+/CD1a+/CD34+
  - 胞质CD3+/CD7+/CD4+/CD8+
  - 胞质CD3+/CD7+/CD13+和/或CD33+

- **常用MRD标志物组合**：
  - 胞质CD3/CD7/表面CD3/CD5
  - 胞质CD3/CD7/CD1a/CD34
  - 胞质CD3/CD7/CD4/CD8
  - 胞质CD3/CD7/CD13/CD33

## 免疫表型与预后的关系

### B-ALL预后相关标志物
- **预后良好相关标志物**：
  - CD10阳性
  - CD20阳性(ETV6-RUNX1阳性ALL)
  - CD38强表达

- **预后不良相关标志物**：
  - CD10阴性
  - CD25阳性(BCR-ABL1阳性ALL)
  - CD56阳性
  - CD123强表达
  - CD34+/CD38-白血病干细胞比例高

### T-ALL预后相关标志物
- **预后良好相关标志物**：
  - CD1a阳性(皮质T-ALL)
  - CD4和CD8双阳性

- **预后不良相关标志物**：
  - CD1a阴性
  - CD13和/或CD33阳性(ETP-ALL)
  - CD56阳性
  - CD34+/CD7+白血病干细胞比例高

## 免疫表型鉴别诊断

### 与急性髓系白血病(AML)的鉴别
- ALL通常MPO阴性，TdT阳性
- B-ALL表达CD19、CD79a、PAX5等B系标志物
- T-ALL表达胞质CD3、CD7等T系标志物
- 混合表型急性白血病需要特殊鉴别

### 与淋巴母细胞淋巴瘤的鉴别
- 淋巴母细胞淋巴瘤和ALL免疫表型相似
- 需结合临床表现和骨髓浸润程度鉴别
- 淋巴母细胞淋巴瘤骨髓浸润<25%

### 与成熟B/T细胞肿瘤的鉴别
- 成熟B细胞肿瘤通常TdT阴性，表面免疫球蛋白阳性
- 成熟T细胞肿瘤通常TdT阴性，表面CD3阳性
- 成熟B/T细胞肿瘤通常CD34阴性
